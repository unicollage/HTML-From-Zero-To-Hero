## ðŸ”¹ What are Table Columns?

* A **column** is a vertical set of cells
* Defined by placing `<td>` or `<th>` inside rows

**Example Table:**

```html
<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Ali</td>
    <td>25</td>
  </tr>
  <tr>
    <td>Aisha</td>
    <td>23</td>
  </tr>
</table>
```

**Output:**

| Name  | Age |
| -- |  |
| Ali   | 25  |
| Aisha | 23  |
